# temperature-monitoring-system
Embedded C based temperature monitoring project
his project was completed as part of academic coursework at P.K. University, Shivpuri.

## Project Overview
- Designed a temperature monitoring system using a microcontroller
- Interfaced temperature sensor and performed ADC reading
- Displayed temperature data through serial communication

## Tools & Technologies
- Embedded C
- Microcontroller
- Temperature Sensor
- ADC
- Serial Communication

Source code is not available as this was an academic guided project.
